# 移动APP基础框架

## 目录结构
- components <br>
    自定义组件 (推荐使用同名文件夹, 即可全局使用)
- libs <br>
    一些工具
- node_modules <br>
    依赖
- pages <br>
    存放页面文件
- static <br>
    静态文件, 如图片等
- store <br>
    vuex 全局状态管理
- unpackage <br>
    打包生成文件
- App.vue <br>
    主页面入口
- main.js <br>
    主程序入口
- package.json <br>
    依赖配置 <br>
- pages.json <br>
    页面配置文件
- uni.scss <br>
    全局样式
